In-browser interface

You can access the Burp Proxy in-browser interface by visiting http://burp with your browser, or by entering the URL of your Proxy listener, for example: http://127.0.0.1:8080.

You can download a copy of your Burp CA certificate, which you can install in your browser to eliminate SSL errors.

You can disable the in-browser interface if required, in the Proxy options.
