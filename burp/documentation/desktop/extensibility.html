Extensibility

You can extend Burp Suite's capabilities in various ways:

  • You can use the BApp Store to install Burp extensions that have been written by users of Burp Suite.
  • You can create your own Burp extensions using the Burp Extender API.
  • You can use Burp's REST API to integrate other tools with Burp Suite.
