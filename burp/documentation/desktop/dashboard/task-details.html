Task details window

Clicking on the pop-out icon for an individual task on the Burp Dashboard opens the task details window. This displays full details of the selected task.

Depending on the type of the task, the details window might include:

  • A summary of the task configuration, progress, and time remaining.
  • Metrics for requests made, network errors, and issues reported.
  • For audit tasks, the issue activity log and audit items view.
  • The event log for the specific task.
